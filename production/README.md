# production

Cartella di lavoro per gli output di fabbricazione generati da KiCad / dal
plugin JLCPCB (Gerber, drill, BOM, CPL, zip per il fab).

Il contenuto e **ignorato dal versioning** (vedi `.gitignore`), perche
rigenerabile dal progetto. Le release ufficiali e riproducibili vanno
pubblicate come **GitHub Releases** (zip allegato) o committate a mano in una
cartella dedicata quando si vuole congelare una revisione del PCB.
